# ping-pong-game

A simple two-player arcade game built with Python and Pygame.

## Description

Ping Pong is a fast-paced paddle game where two players compete to hit a bouncing ball past each other's paddle. The first player to score 10 points wins.

## Features

- Two-player local multiplayer
- Real-time paddle and ball physics
- Progressive ball speed increase
- Minimal, clean visual design
- Instant restart after match ends

## Controls

| Player | Move Up | Move Down |
|--------|---------|-----------|
| Player 1 (Left) | W | S |
| Player 2 (Right) | Up Arrow | Down Arrow |

- **R** — Restart game
- **ESC** — Quit

## Requirements

- Python 3.x
- Pygame library
